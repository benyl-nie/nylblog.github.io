Jan 13, 2020

项目升级 bootstrap转antd调研

背景

团队内部现有UI库为antd，现升级react 至 16，UI库为bootstrap和react-bootstrap， 考虑升级成本与替换成本差不多，现计划将BD和BDAdmin的UI库统一为antd

项目中用到的常用组件

  • Button
  • Modal
  • Table
  • Tabs
  • Tab
  • SplitButton
  • MenuItem
  • Pagination
  • Tooltip
  • OverlayTrigger
  • react-select
  • react-simple-radio-group

细化

以下会对上述项目中所用到的组件逐个对比，评估迁移成本与解决方案

  • Button 按钮

- react-bootstrap 0.27.3 antd 3.22.1
引入方式 import Button from ‘react-bootstrap/lib/Button’; import { Button } from ‘antd’;
属性 disabled,ghost,href, htmlType, icon, loading, shape, size,target, type,onClick,block

完成情况： done
